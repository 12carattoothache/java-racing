# 자동차 경주 게임

## 목적

- java 를 활용한 클린 코드를 배워봅니다.

## 환경 

- java 11 / junit 5를 기준으로 합니다.

## 진행 방법

* 차수에 맞추어 과제를 수행합니다.
* 과제를 완료한 뒤에 자신의 github 아이디에 해당하는 브랜치에 PR을 요청합니다. 
* 리뷰 피드백을 받고 코드를 개선하여 다시 push 합니다.
* PR이 approve 되면 코드를 merge 하고 다음 과제를 수행합니다.
